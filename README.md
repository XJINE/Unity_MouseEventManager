# Unity_MouseEventManager

<img src="https://github.com/XJINE/Unity_MouseEventManager/blob/main/Screenshot.png" width="100%" height="auto" />

MouseEventManager provides a mouse button based event handling system.

## Importing

You can use Package Manager or import it directly.

```
https://github.com/XJINE/Unity_MouseEventManager.git?path=Assets/Packages/MouseEventManager
```

### Dependencies

This project use following resources.

- Unity.InputSystem

## How to Use

Register your button and function pairs in the Inspector.
